# Flags

Only current regional and territorial flags. For a flag to show as an option, it muse be included in `modules/user/src/main/Countries.scala`.

Flags were taken from https://gitlab.com/catamphetamine/country-flag-icons version 1.6.4, not all flags are necessarily used.

## Following changes or additions were made:

### Changes:

- CH - Square
- VA - Square, coat of arms from twemoji
- US - Colors and stars from twemoji
- NP - Square, Make background transparent
